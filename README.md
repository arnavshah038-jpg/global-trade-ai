 📈 Global Trade & Geopolitical AI Analyzer

An AI-powered financial dashboard that analyzes global trade news using Natural Language Processing (NLP) to provide actionable stock market insights.

 🚀 Key Features
- Live News Ingestion: Fetches real-time commodity and trade news using RSS feeds (CNBC, Reuters).
- Sentiment Analysis: Uses `TextBlob` to calculate the polarity of news headlines.
- Stock Correlation: Automatically maps news keywords to specific Indian stock sectors (Sugar, Energy, Agri, Logistics, etc.).
- Trade Signals: Generates Buy/Hold/Sell recommendations based on impact scores.
- Interactive UI: Built with `Streamlit` for a clean, professional user experience.

 🛠️ Tech Stack
- Language: Python 3.x
- Framework: Streamlit
- Libraries: - yfinance (Stock Data)
  - `TextBlob` (NLP Sentiment)
  - `feedparser` (News Ingestion)
  - `pandas` (Data Handling)
